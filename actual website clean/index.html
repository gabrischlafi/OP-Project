PROJECTS.

TROBBIES.

trobbies is a tattoo studio located in the hague, netherlands. gabriel has worked together with the studio and the owner thomas "trobbies" brouwer.
this project initiated with gabriels research for design portfolio. it all started off with a simple and nice conversation about their goals with this project. after doing his research on the studio he was sure what kind of pictures would fit to the aesthetic of the studio. afterwards they held a 40 minute interview discussing him and his relation to tattooing and captured a 4 hour tattoo session.
the pictures and videos that were taken at the studio were then added to the website and social media page with a following of 29.000 followers.

PROCESSING.

the processing project took place during the programming bootcamp with gabriels university. the students had five days of time to complete 4 projects. Each project should be done and presented after the classes in which they have shown the students the code that we should use in the projects. the purpose of processing is teaching non-programmers the fundamentals of computer programming in a visual context.

the first project was fairly simple code. what gabriel has created are multiple containers that generate on the left one randomly generatet color and one on the right. with a mathematical formula he has come up with the colours start to generate a gradient.


the second project was supposed to be an interactive game. gabriel has decided to create a game seemingly to the well known tamagotchis. a smiley, which you have to feed, give water and love so it doesnt "die". the design was completely self made and done by gabriel.

the third design was for the students to create an interactive artwork. the interactive artwork gabriel has created consists of ellipses that multiply and change colour depending on the positioning of the mouse on the x-axys.

the fourth and last project was for the students to design a project that amazes the teachers. including all the code and ideas the students have collected throughout the bootcamp. gabriel has created a project that looks like a snake. this snake follows the mouse position around. he has integrated ideas from before, sich as the gradient idea. there are two colors randomly selected by the program and throughout the 45 created ellipses it creates a gradientof colors.
